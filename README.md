Machine Learning project for customer segmentation using K-Means Clustering and PCA in Python.
# Customer Personality Segmentation using Machine Learning

## Project Overview

This project segments customers into different groups based on their demographic information and purchasing behavior using K-Means Clustering. Principal Component Analysis (PCA) is applied for dimensionality reduction and cluster visualization.

## Problem Statement

The objective of this project is to identify distinct customer segments that can help businesses understand customer behavior and support targeted marketing strategies.

## Dataset Information

The dataset contains customer information such as:

- Age
- Income
- Education
- Marital Status
- Number of Children
- Spending on Various Products
- Purchase Channels
- Campaign Responses
- Customer Engagement

The project uses unsupervised learning, so there is no target variable.

## Results

The K-Means model groups customers into meaningful clusters based on their purchasing patterns and demographic characteristics. PCA is used to visualize the customer segments in two dimensions.
